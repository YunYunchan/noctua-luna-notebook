# Noctua Luna — 日々の記録帳

ひすいこたろう「しあわせがずっと続く手帳」から引き継いだ3つの要素(その日の出来事 / My Happy・Your Happy / 予祝)を記録するジャーナリングアプリ。

- ビルド不要の素の HTML / CSS / JS(フレームワークなし)
- データはブラウザの `localStorage` にのみ保存。サーバーには一切送信されません
- 新月・満月の日付は自動計算(平均朔望月ベースの近似値)

## ローカルで開く

ビルド不要なので、`index.html` を直接ブラウザで開くだけで動作します(ダブルクリックでOK)。

## 公開する(GitHub Pages / Netlify / Vercel)

このフォルダ一式(`index.html` / `css/` / `js/`)をそのまま静的ホスティングにアップロードするだけで公開できます。

- **GitHub Pages**: リポジトリの Settings → Pages でこのフォルダを公開
- **Netlify / Vercel**: リポジトリを連携するだけ(ビルドコマンドは不要、公開ディレクトリはルート)

友達に配る場合は、公開したURLを共有するだけでOKです。各自のブラウザにそれぞれ独立してデータが保存されるため、お互いの記録が見えることはありません。

## データのバックアップ

機種変更やブラウザのデータ削除で記録が消えてしまうため、「設定」タブの「データをエクスポート」で定期的にJSONバックアップを取ることをおすすめします。同じ画面の「データをインポート」で復元できます。
